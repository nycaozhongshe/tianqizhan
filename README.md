# weather-app
---  

## 前言  

基于 vue2.0 写的单页面应用 weather-app，所有的数据来自国外的一个API: https://openweathermap.org/api。

## 技术栈

- vue2.0
- vue-cli
- vue-router
- resources
- Sass

## 实现功能

- 天气的的具体展示
- 城市天气查询
- 添加和删除城市记录

## 执行项目

> 安装依赖  
> npm install  
> 启动服务  
> npm run dev  
> 运行  
> localhost:8080/#/Layout/Home
